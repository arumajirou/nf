# レベル別レポートテンプレート

トレンド資料を作成するときの「レベル0〜2」の基本テンプレートです。
必要に応じて見出し名や順番を調整して構いませんが、
一貫性はできるだけ保ってください。

---

## レベル0：全体サマリページ テンプレート

```markdown
# 今日のAI / Python トレンドまとめ（YYYY-MM-DD）

## 1. 一言まとめ（TL;DR）

- 例: 軽量LLMとRAG関連のライブラリが急増している日
- 例: 画像生成系のSpacesが活発  など

## 2. 今日よく目にするキーワード

- キーワードA: 一言説明
- キーワードB: 一言説明
- キーワードC: 一言説明

## 3. 情報源マップ

- GitHub Python トレンド: 「コードのトレンド」を知る場所
- HF 論文トレンド: 「研究の最先端」を知る場所
- HF モデル: 「すぐ使えるAIエンジンの棚」
- HF データセット更新: 「学習材料の倉庫」
- HF スペース更新: 「触って試せるデモ広場」

（必要ならここに Mermaid の mindmap / flowchart を追加）

## 4. 今日の注目カテゴリ

- カテゴリA（例: GitHub）: 一言でいうとどんな日か
- カテゴリB（例: モデル）: 一言でいうとどんな日か
- カテゴリC（例: スペース）: 一言でいうとどんな日か

## 5. 次のステップの提案

- 完全初心者の人は：○○のセクションから読むとよい
- ある程度慣れている人は：○○の一覧表を中心に見るとよい
- 実務で使いたい人は：○○のモデル/データセットを確認するとよい
```

---

## レベル1：カテゴリ別サマリページ テンプレート

例として「GitHub Python トレンド」用テンプレートを示します。
他のカテゴリ（論文 / モデル / データセット / スペース）も同様の構成を使えます。

```markdown
# GitHub Python トレンドまとめ（YYYY-MM-DD）

## 1. このページでわかること

- 今日の Python 関連リポジトリの「人気どころ」をざっくり掴めます。
- 自分のレベルに合った「試してみる候補」を見つけられます。

## 2. 誰向けか

- Python を勉強し始めた人
- 機械学習・データサイエンスのサンプルコードを探している人
- 新しいツールをキャッチアップしたいエンジニア

## 3. 今日のトップトレンド一覧

（ここにカテゴリ別テーブルを1つ挿入：詳細は別紙テンプレート）

## 4. どう活用するとよいか（3ステップ）

1. 一覧から「目的が近いもの」を1〜2件選ぶ
2. README を読み、サンプルコードを1つ実行してみる
3. 気に入ったらブックマークやメモに追加する

## 5. よくあるつまずきと対策

- Python のバージョンが合わない → 対処方法の例
- GPU が必須のプロジェクト → 代わりに試せる軽量プロジェクト例

## 6. もっと知りたくなった人向けリンク

- GitHub 公式ドキュメント
- Python 入門コース / チュートリアル など
```

## レベル2：ヘッドライン詳細ページ テンプレート

個々のトレンド（リポジトリ / 論文 / モデル / データセット / スペース）について、
詳しく解説するときのテンプレートです。

```markdown
# ヘッドライン詳細：{{名前}}

## 1. これは何か

- 一言でいうと: ◯◯
- もう少し詳しく: 2〜3行で説明

## 2. どんな場面で役立つか（ユースケース）

- ユースケース1：説明
- ユースケース2：説明
- ユースケース3：説明（あれば）

## 3. どうやって触るか（最短チュートリアル）

1. 環境準備（例: Python 3.10 / 必要なライブラリ）
2. サンプルコードまたはデモのURLを試す手順
3. もう一歩進めたい人向けの次の手順

## 4. 強みと弱み

| 観点 | 強み | 弱み / 注意点 |
|------|------|----------------|
| 導入のしやすさ | 例: pip で簡単に入る | 例: GPU が必要 |
| 性能 / 機能 | 例: 同種の中では高速 | 例: 特定のタスクには向かない |
| 学習コスト | 例: 公式チュートリアルが充実 | 例: 英語の情報が多い |

## 5. 似たものとの違い

- 類似プロジェクトAとの違い
- 類似プロジェクトBとの違い（あれば）

## 6. どのレベルの人向けか

- 難易度の目安: 入門 / 中級 / 上級
- 向いている人の例

## 7. 次に一歩進むなら

- 応用アイデア1
- 応用アイデア2
- 関連プロジェクト / 論文 へのリンク
```

このテンプレートをベースに、ユーザーのリクエストに応じて
必要なセクションだけ使う・順番を変えるなどのカスタマイズを行ってください。
