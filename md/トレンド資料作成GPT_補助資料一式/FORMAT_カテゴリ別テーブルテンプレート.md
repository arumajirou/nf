# カテゴリ別テーブルテンプレート

各トレンドカテゴリごとの「一覧表」のテンプレートです。
実際に使うときは、必要なカラムだけ残しても構いません。

---

## 1. GitHub Trending / Python 用

```markdown
| ヘッドライン | 分類タグ | 主な機能 | 想定ユーザー | 活用シーン | 人気指標 | なぜトレンド？ | 難易度 | リンク |
|--------------|----------|----------|--------------|------------|----------|----------------|--------|--------|
| 例: repo-name | 例: Web, ML | 何をしてくれるリポジトリか | 初心者 / 中級MLエンジニア など | 具体的な利用例 | Star数, Fork数 など | シンプルさ / 性能 / 話題性 など | 入門 / 中級 / 上級 | GitHub URL |
```
